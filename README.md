# AI-помощники в Cursor: карьера, здоровье, инвестиции

## Что внутри

- **`career-compass`** — карьерный компас, разбор вакансии и ревью CV. Данные: папка `career/`.
- **`health-checkup`** — разбор анализов с уклоном в превентивную медицину. **Не заменяет врача.** Данные: папка `health/`.
- **`stock-analyst`** — разбор публичных компаний по ссылке на investor relations. **Не инвестиционная рекомендация.** Данных в проекте не нужно.

## Как запустить

1. Установите Cursor: https://cursor.com.
2. Распакуйте архив.
3. В Cursor: **File → Open Folder** → выберите эту распакованную папку. Все три ассистента подтянутся сами.
4. Откройте чат справа и начните писать. Примеры ниже.

Бесплатного плана Cursor хватит попробовать. Для серьёзной работы нужен Pro ($20/мес) — иначе модели и контекст быстро режутся лимитами.

## Куда класть свои файлы

- **CV** (PDF / md / txt, ru/en) → `career/cv/`.
- **Анализы** (PDF / сканы / фото, ru/en/de) → `health/labs/`.
- **Карьерный профиль** — заполните `career/profile.md` (чем больше — тем меньше вопросов в чате).
- **Медицинский анамнез** — заполните `health/profile.md`. Что не знаете — пишите «не указано», ассистент спросит.

Для инвестиций ничего класть не нужно: достаточно ссылки на раздел investor relations прямо в чате.

## С чего начать разговор

- **Карьера**: «Разбери эту вакансию: {ссылка}. Смогу ли я туда пройти?» / «Посмотри моё CV под роль X» / «Хочу карьерный компас».
- **Здоровье**: «Пройди со мной чекап по моим анализам» / «Сравни новые результаты со старыми» / «Что мне стоит досдать в этом году?».
- **Инвестиции**: «Стоит ли сейчас заходить в {компания}? IR: {ссылка}».

## Приватность

- Всё работает локально в Cursor. Ваши файлы никуда не заливаются автоматически — модель видит только то, что вы подключили в контекст.
- `career/cv/`, `health/labs/`, `profile.md` и `summary.md` — личные. Не комитьте их в публичные репозитории.

## Дисклеймеры

- **Здоровье.** Ассистент не ставит диагнозы, не назначает препараты и не заменяет очный приём. Он помогает систематизировать данные и подготовить вопросы врачу.
- **Инвестиции.** Ничто в этом проекте не является инвестиционной рекомендацией. Решения и риски — ваши.
