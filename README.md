# AI-ассистенты в Cursor: карьера, здоровье, инвестиции

Набор [Agent Skills](https://docs.cursor.com/context/skills) для Cursor: три тематических помощника и папки под ваши данные.

## Быстрый старт

1. Установите [Cursor](https://cursor.com) (Windows, macOS, Linux).
2. Склонируйте репозиторий и откройте папку в Cursor:

   ```bash
   git clone https://github.com/shigirdanova/ai-tools-guten-anna.git
   cd ai-tools-guten-anna
   ```

   В Cursor: **File → Open Folder…** → выберите `ai-tools-guten-anna`.

3. Откройте чат (**Cmd+L** / **Ctrl+L**) и пишите по-русски или по-английски — скиллы подхватываются из `.cursor/skills/` автоматически, отдельная установка не нужна.


## Что внутри

| Скилл | Назначение | Данные в репо |
| --- | --- | --- |
| [**career-compass**](.cursor/skills/career-compass/SKILL.md) | Карьерный компас, разбор вакансии, ревью CV | папка [`career/`](career/) |
| [**health-checkup**](.cursor/skills/health-checkup/SKILL.md) | Разбор анализов, превентивный уклон, вопросы к врачу **не заменяют приём** | папка [`health/`](health/) |
| [**stock-analyst**](.cursor/skills/stock-analyst/SKILL.md) | Разбор публичных компаний по IR и отчётности **не инвестиционная рекомендация** | файлы в репо не обязательны |

---




## Куда класть свои файлы

- **CV** (PDF, md, txt; ru/en) → `career/cv/`
- **Анализы** (PDF, сканы, фото; ru/en/de) → `health/labs/`
- **Карьерный профиль** → [`career/profile.md`](career/profile.md) (чем полнее — тем меньше уточняющих вопросов в чате)
- **Медицинский анамнез** → [`health/profile.md`](health/profile.md); неизвестное — «не указано», ассистент доберёт в диалоге

Для блока про инвестиции достаточно ссылки на раздел **Investor Relations** в чате, отдельная папка не нужна.


## Примеры запросов

- **Карьера:** «Разбери вакансию: {ссылка}. Реалистично ли мне туда пройти?» / «Посмотри CV под роль …» / «Хочу карьерный компас».
- **Здоровье:** «Пройди чекап по моим анализам» / «Сравни новые результаты со старыми» / «Что имеет смысл досдать в этом году?»
- **Инвестиции:** «Оцени заход в {компания}, вот IR: {ссылка}».

---



## Дисклеймеры

- **Здоровье.** Ассистент не ставит диагнозы, не назначает лечение и не заменяет очный визит. Цель: структура данных и вопросы к врачу.
- **Инвестиции.** Ничто здесь не является инвестиционной рекомендацией, офертой или призывом к сделке. Решения и риски остаются за вами.
